# 💎 Omar Jewellers

A modern full-stack jewellery e-commerce website built with **Next.js**, **Tailwind CSS**, **Prisma**, and **PostgreSQL (Neon DB)**. The application provides a smooth shopping experience along with an admin panel for managing products and website content.

## ✨ Features

* 🛍️ Browse jewellery products
* 📱 Fully responsive design
* 🛒 Shopping cart
* ⚙️ Admin dashboard
* 🗄️ PostgreSQL database with Prisma ORM
* 🚀 Deployed on Vercel

## 🛠️ Tech Stack

* Next.js
* React
* Tailwind CSS
* Prisma ORM
* PostgreSQL (Neon DB)
* Vercel

🎯 Future Improvements
